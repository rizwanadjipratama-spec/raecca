# Raecca Premium Website (Awwwards Edition)

## 🎨 Aesthetic & Design System
- **Style:** Y2K Coquette, Editorial Magazine, High-Gloss, Glassmorphism
- **Color Palette:**
  - Base: Cream / Off-White (`#FAF8F5`)
  - Primary Accent: Hot Pink / Magenta (`#D11261`)
  - Soft Accents: Pastel Pink (`#FFD1DC`), Peach (`#FFDAB9`)
  - Typography: Deep Burgundy / Dark Charcoal for high contrast.
- **Typography:**
  - Headings: Elegant Serifs (e.g., *Playfair Display* or *Cormorant Garamond*).
  - Body: Clean Sans-serifs (e.g., *Inter* or *Outfit*).
- **Vibe:** Cinematic, Floaty, Premium, "Absolute Cinema".

## 🚀 Tech Stack
- Framework: Nuxt 3 (Vue.js)
- Styling: Tailwind CSS
- Animation: GSAP (ScrollTrigger, Timeline)
- Smooth Scroll: Lenis
- Icons: Lucide Vue Next
